# 아트 팜 - Art Farm (2021.12 ~ 2022.01)
> 아마추어(또는 무명)아티스트들을 위한 전시회 소개 웹 서비스

<br>

## 주요 기능
* 온라인 전시자료 업로드
* 온/오프라인 전시정보 제공


## 사용 기술
* Java 8
* Spring Boot 2.6.2
* Gradle 7.3.1
* Spring Data JPA
* Vue.js
